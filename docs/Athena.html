1. 产品定位1. Product Positioning

Athena 不是单一聊天机器人，而是 Santoni 面向无缝针织价值链的智能工作层：连接品牌设计、Arachne 参数化定制、SWS 高级打样、OEM 工厂接单决策、售后服务和行业情报。

Athena is not a standalone chatbot. It is Santoni's intelligent work layer for the seamless knitting value chain, connecting brand design, Arachne parameterized customization, SWS advanced sampling, OEM order decisions, service operations, and industry intelligence.

设计到生产Design to Production

把自然语言、参考图和参数选择转成款式、纱线、结构、3D 静态展示和生产文件方向。

Convert natural language, references, and parameter choices into style, yarn, structure, static 3D previews, and production-file direction.

订单 ROIOrder ROI

帮助 OEM 老板判断订单要不要接、能不能接、接了之后收益和风险如何。

Help OEM decision makers assess whether to accept an order, whether capacity is sufficient, and what return and risk it creates.

服务运营Service Operations

通过服务案例、在线排查、受控工具和派工准备，提高售后响应效率。

Improve service response through case knowledge, online assistance, controlled tools, and dispatch preparation.

行业情报Industry Intelligence

定期监测展会、竞品、客户、政策、技术和海外机会，并转化成行动建议。

Monitor exhibitions, competitors, customers, policy, technology, and overseas opportunities, then convert them into actions.

5.1 PMO Agent / Project Intelligence Agent5.1 PMO Agent / Project Intelligence Agent

PMO Agent 是 Athena 的内部管理型子 Agent，不面向客户销售。它的目标是让 Athena 自己的研发、商业化、数据准备和试点进度变得可计算、可追溯、可预警，而不是依赖人工主观填写百分比。

PMO Agent is Athena's internal management sub-agent, not a customer-facing commercial product. Its goal is to make Athena's engineering, commercialization, data preparation, and pilot progress calculable, traceable, and alertable instead of relying on subjective manual percentages.

证据来源Evidence Sources

任务系统、验收 checklist、Git/PR/CI、文档 Change Log、会议纪要、管理层决策、试点客户反馈。

Task systems, acceptance checklists, Git/PR/CI, document change logs, meeting notes, management decisions, and pilot customer feedback.

计算引擎Calculation Engine

用确定性规则和权重计算模块进度、延期状态、风险等级和缺失证据；LLM 只负责总结和解释。

Uses deterministic rules and weights to calculate module progress, delay status, risk level, and missing evidence; LLM only summarizes and explains.

输出动作Outputs

项目周报、风险预警、需要 Danny / Belotti 或跨部门确认的决策清单、owner 提醒和下一步建议。

Project weekly reports, risk alerts, decision lists for Danny / Belotti or cross-team review, owner reminders, and next-action suggestions.

6.3 Smart Machine / Smartex 与预测性维护6.3 Smart Machine / Smartex and Predictive Maintenance

Santoni DT 对大圆机 Smart Machine 的方向，是在机器上增加传感器和边缘处理能力，让机器从“被动报警”升级为“自我监测、诊断和逐步自我修复”。当前应拆成两个项目管理：Smartex 智能验布作为短期商业化亮点；Predictive Maintenance 作为独立 P2 技术项目，先补齐振动采集、信号处理和数据闭环。

Santoni DT's Smart Machine direction for large-diameter machines is to add sensors and edge processing so machines evolve from passive alarms toward self-monitoring, diagnosis, and eventually guided self-recovery. It should now be managed as two projects: Smartex vision inspection as the near-term commercial highlight, and Predictive Maintenance as a separate P2 technical project focused first on vibration acquisition, signal processing, and data loops.

Smartex 定位Smartex Positioning

Smartex 智能验布是视觉 AI 产品化问题：在生产过程中完成验布动作，并对布匹质量进行 inspection 与分类。

Smartex vision inspection is a vision-AI productization problem: inspect fabric during production and classify roll quality.

核心优势Core Advantage

相较单摄像头上方安装方案，Smartex 在 take-down 安装 8 个摄像头，并结合可见光、红外、紫外三种波段，目标最高分辨率 1mm。

Compared with single top-camera inspection, Smartex uses eight take-down cameras with visible, infrared, and ultraviolet bands, targeting up to 1 mm resolution.

真正护城河Real Moat

优势不只是摄像头数量，而是能和机台速度、take-down 张力、订单、纱线、组织、报警和缺陷位置绑定，形成织造过程质量感知系统。

The moat is not camera count alone, but the ability to bind defects to speed, take-down tension, order, yarn, structure, alarms, and machine position.

7. APS 排单与利润率提升7. APS Scheduling and Margin Improvement

The glossary aligns bilingual terminology across sales, product, R&D, service, and business teams.

English 中文 说明Explanation
Knitting针织用织针把纱线弯曲成线圈并互相串套形成织物。Fabric formation by bending yarn into intermeshed loops with needles.
Weft knitting纬编纱线沿横向形成线圈横列，常见于圆机、横机、无缝针织。Yarn forms courses horizontally; common in circular, flat, and seamless knitting.
Warp knitting经编多根经向纱线分别供给织针，常用于网布、蕾丝、运动面料。Multiple warp yarns feed needles lengthwise; common in mesh, lace, and sports fabrics.
Gauge机号 / 针距机器语境中常指每英寸针数，影响可织纱线和布面细密程度。Usually needles per inch in machine context; affects yarn fineness and fabric density.
Stitch length线圈长度单个线圈消耗的纱线长度，影响克重、手感、弹性和尺寸稳定性。Yarn length consumed by one loop; affects weight, hand feel, elasticity, and stability.
Covered yarn包覆纱以氨纶等为芯纱，外层用其他纱线包覆，常用于袜子、无缝、运动服。Core yarn such as elastane covered by other yarns; used in socks, seamless, sportswear.
Jacquard提花通过选针形成图案、纹理或颜色变化。Pattern, texture, or color effects created through needle selection.
Mesh网眼布形成孔眼结构，常用于透气运动服、鞋面、内衬。Open-hole structure used for breathable sportswear, uppers, and lining.
Plating添纱 / 衬垫编织多根纱在同一线圈中按正反位置排列，可实现内外层不同性能。Multiple yarns positioned in the same loop to create different inner/outer properties.
Needle织针形成线圈的核心部件。Core component used to form loops.
